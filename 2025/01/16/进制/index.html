10进制和其他进制互转

strtoul 其他进制转10进制

  • 利用函数strtoul(const char *, char **, int)，通过对应的进制格式，将字符串转换成十进制

    第一个参数为需要进行转换的目标字符串，
    第二个参数为转换过程中终止的位置，
    第三个参数为与目标字符串对应的进制，置0表示自动检测转换格式，
    例：代码 ‘FFt6’ 为目标字符串，FF为十六进制，第三个参数置为16。

执行结果中第三个参数为第一个不能转换的字符：’t’,十六进制的’FF’对应十进制的225;
top：该方法通常用到将其他进制字符串转换成十进制数。

常用格式化符号解释

占位符 描述 示例输入 示例输出
%d 以 有符号十进制整数 输出（signed int）。 -42 -42
%u 以 无符号十进制整数 输出（unsigned int）。 42 42
%x 以 无符号十六进制整数（小写字母） 输出。 255 ff
%X 以 无符号十六进制整数（大写字母） 输出。 255 FF
%o 以 无符号八进制整数 输出。 255 377
%f 以 浮点数（小数点） 格式输出，默认显示 6 位小数。 3.14 3.140000
%.2f 控制小数位数的浮点数输出，2 表示保留两位小数。 3.14 3.14
%e 以科学计数法（小写 e）格式化浮点数。 314.15 3.1415e+02
%E 以科学计数法（大写 E）格式化浮点数。 314.15 3.1415E+02
%c 输出 单个字符。 ‘A’ A
%s 输出 字符串。 “Hello” Hello
%p 以 指针（内存地址） 的形式输出。 ptr 0x7ffee…
%% 输出一个百分号 % 本身。 无 %
%lld 以 有符号长长整型（long long int） 格式输出。 -12345 -12345
%llu 以 无符号长长整型（unsigned long long int） 格式输出。 12345 12345
%zd 按目标平台的 size_t 类型格式输出，通常用于打印数组长度或指针差值。 42 42
%zu 按目标平台的无符号 size_t 类型格式输出。 42 42

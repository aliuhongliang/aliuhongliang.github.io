var c = Circle()
【存储属性】c的地址值就是 和c中width的地址是一样的：width就是在c的第一个位置， 指针指的一个地方。结构体第一个属性的地址值就是结构体变量的地址值。
所以当传递 &c.width 的时候，实际传递的就是c的地址值。
总结：普通存储属性传递的就是该属性的引用地址。

【计算属性】结构体中计算属性test，因为它不占内存，所有不可能传递test的内存，具体实现：首先调用get方法，将值存入栈上一个临时地址，将这个临时地址传递给 inout，然后修改临时地址的值，最后调用set方法。

【有观察器的存储属性】传递带属性观察的存储属性例如side。&c.side,会将side存储的值，存到栈上一个临时地址，将这个临时地址传递给 inout，在方法中修改这个临时地址的值，函数调用完成后，将临时地址的值赋值给side，完成修改。使用临时地址是为了触发willset和didset

protocol TestProtocol {
func test1()
}

extension TestProtocol {
func test1() { }
func test2() { }
}

class TestClass: TestProtocol {
func test1() { }
func test2() { }
}

var cls = TestClass()
cls.test1()
cls.test2()

这里的cls 是TestClass 类型，调用方法时候，首先看协议中定义的方法，优先去 TestClass 中找方法的实现，没有就去类的扩展中找实现，还没有就去协议的扩展中找方法的实现。如果调用的方法没有在协议中定义，那么直接去类、类的扩展中找方法的实现
